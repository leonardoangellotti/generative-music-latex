# Generative Music

Project focused on generative music algorithms and LaTeX-based documentation for experiments, analysis, and results.

## Contents

- Generative music experiments
- Algorithmic composition
- LaTeX reports and documentation
- Audio and visualization outputs

## Structure

- `src/` — source code and generation scripts
- `latex/` — LaTeX documents and reports
- `output/` — generated music and results
- `assets/` — figures and supporting material

## Usage

Compile the LaTeX documents with:

```bash
pdflatex main.tex
